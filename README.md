# 🍫 Chocolate Sales Dashboard — Power BI

An interactive Power BI dashboard analysing chocolate product sales 
across multiple countries, built to surface business insights from raw sales data.

---

## 📌 Project Overview

This dashboard helps business teams understand which chocolate products 
are generating the most revenue, how sales vary across geographies, 
and which products need attention — all through interactive visuals.

---

## 📊 Dashboard Features

| Visual | Description |
|---|---|
| Bar Chart | Sum of Amount by Product — ranks all chocolate SKUs by revenue |
| Geo Filter | Filter sales data by country — Australia, Canada, India, New Zealand, UK, USA |
| Product Comparison | Side-by-side revenue comparison across 20+ chocolate products |

---

## 💡 Key Insights

- **Top performing product:** Manuka Honey & Dark Choco generated the highest revenue
- **Geography:** Dashboard supports multi-country filtering for regional analysis
- **20+ SKUs** analysed including Milk Bars, Organic Choco, Almond Choco, and more
- Helps identify **slow-moving products** vs **high-revenue SKUs** for business decisions

---

## 🛠️ Tech Stack

- **Tool:** Microsoft Power BI Desktop
- **Data:** Chocolate sales dataset (Excel/CSV)
- **Concepts:** Data modeling, DAX measures, interactive filtering, bar chart visualisation

---

## 🚀 How to View

1. Download the `power bi {data modeling} choclate.pbix` file
2. Open with **Microsoft Power BI Desktop** (free download from Microsoft)
3. Interact with the filters to explore sales by country and product
